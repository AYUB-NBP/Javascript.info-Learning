let user = new Object(); // "object constructor" syntax
let user = {}; // "object literal" syntax

When we use {}to create an object. That declaration is called an Object literal.

Literals and properties

We can immediately put some properties into {...} as “key: value” pairs:

let human = { name: "Ayoub", age: "25", IQ: "999" };

 

• We can access any property value from an object using the object' s declared name and a key within it.

Square brackets

• For multiword properties, the dot access doesn’t work: ( So we use Square Brackets)

The dot requires the key to be a valid variable identifier. That implies: contains no spaces, doesn’t start with a digit and doesn’t include special characters ($ and _ are allowed).

let user = {};

// set
user["likes birds"] = true;

// get
alert(user["likes birds"]); // true

// delete
delete user["likes birds"]; Check logs

Note: Key is properly "quoted" within the square brackets.

• There is an advantage to using string notation as it allows us to use expressions that define the property key we can pass to retrieve a value. That gives us a lot of flexibility as the key could be an input from the user

Example:

let user = {
name: "John",
age: 30
};
// access by variable
function inputKey() {
let key = prompt("What do you want to know about the user?", "name");
alert(user[key])
}; // John (if enter "name")

• The dot notation cannot be used in a similar way:

let user = {
name: "John",
age: 30
};

let key = "name";
alert( user.key ) // undefined

Computed properties

We can use square brackets in an object literal, when creating an object. That’s called computed properties.

let game = prompt("Which game to buy?", "Squad");

let purchase = {
[game]: '5$', // the name of the property is taken from the variable game };

alert('The price of ' + game + ' is ' + purchase.Squad); // 5 if game="Squad"  

The meaning of a computed property is simple: [game] means that the property name should be taken from game. So, if a visitor enters "Squad", purchase will become {Squad: '5$'}.

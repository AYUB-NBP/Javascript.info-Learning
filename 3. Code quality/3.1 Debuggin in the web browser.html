Debugging in the browser

Debugging is the process of finding and fixing errors within a script.

Console

Easy stuff, you press F12 and get to the console and also get to see files in the Sources tab.

You can press Esc to open a Console panel if you're not on the console Tab already

Breakpoints

Apparently in the Sources tab when the dev tools, You can choose within the .Js file break points where the code execution can be automatically paused at those points.

While the code is paused, we can examine current variables, execute commands in the console etc. In other words, we can debug it.

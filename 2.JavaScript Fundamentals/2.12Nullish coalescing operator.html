Nullish coalescing operator '??'

?? returns the first defined argument.

This:

result = a ?? b

is equivalent to this:

result = (a !== null && a !== undefined) ? a : b

Provided "a" is defined, both lines of code should return "a"; The first defined argument.

Examples of use for Nulish Coalesching operator:

  1. Provides a default value in case the main argument is not defined.

    let user;

    alert(user ?? "Anonymous"); // Anonymous (user not defined)

  2. Finds the first defined argument amongst many:

    3. let firstName = null;
    let lastName = null;
    let nickName = "Supercoder";

    // shows the first defined value:
    alert(firstName ?? lastName ?? nickName ?? "Anonymous"); // Supercoder
?? and || do the same thing but the important difference between them is that:
  • || returns the first truthy value.
  • ?? returns the first defined value.

• The operator ?? has a very low precedence, only a bit higher than ? and =, so consider adding parentheses when using it in an expression.

• It’s forbidden to use it with || or && without explicit parentheses.

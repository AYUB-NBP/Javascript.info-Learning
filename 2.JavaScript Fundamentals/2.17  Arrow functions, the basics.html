Arrow functions, the basics

Arrow function syntax is better than Function Expressions.

let func = (arg1, arg2, ...) => //returned code

Example:

let multiply = (a, b) => a + b;
multiply(2, 2); // 4. 

Multiline Arrow Function

Arrow functino with more complex returned result

Example:

// math function a*b+c
let mathFunc = (a, b, c) => {
let multiplication = a * b;
let addition = multiplication + c;
return addition;
}

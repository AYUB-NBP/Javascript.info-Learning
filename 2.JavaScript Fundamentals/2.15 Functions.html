• Local and Global Variables

    Local Variables

Basically any variable used by the function that's inside of it. It's inaccessible from outside the function. Great way to isolate daat without problems with other functions.

    Global Variables

Variables declared outside of any function, such as the outer userName in the code above, are called global. Global variables are visible from any function (unless shadowed by locals). It’s a good practice to minimize the use of global variables. Modern code has few or no globals. Most variables reside in their functions. Sometimes though, they can be useful to store project-level data.

• Parameters

  • A parameter is the variable listed inside the parentheses in the function declaration (it’s a declaration time term).
  • An argument is the value that is passed to the function when it is called (it’s a call time term).

Example:

function WhatILike(fruit, veggy) {
alert('I like ' + fruit + ' and ' + veggy + '!');

After clicking this button, the function will be called with these arguments:

WhatILike(`Bananas`,`Tomatoes`)

• Default values

When creating a function and you put parameters in it. A value can be assigned to those parameters that will be default to when no argument has been used the function is called.

Example:

let x, y;
function XY(x = "0", y = "0") {
alert(x + ', ' + y);
}
